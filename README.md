## Single Page Website using zurb foundation 6, scroll magic and GSAP 

Use the following commands to clone and run the code:

Clone this repo

```git clone https://github.com/gigacoder/zurb-scrollmagic-gsap.git```

Change the directory

```cd zurb-scrollmagic-gsap```

Run npm install to install node dependencies mentioned on package.json

```npm install```

Run bower install to install bower components mentioned on bower.json

```bower install```

Start the node server

```npm start```
